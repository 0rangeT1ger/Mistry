# Mistry
这个网页使用了Node搭建了临时服务器，使用了React框架。
运行时，打开目录,然后输入
	$ npm install
敲下回车。
等待安装完成，输入
	$ node startServer
按下回车，然后打开浏览器访问localhost:3000就可以看到网页。
